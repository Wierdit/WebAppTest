# CLAUDE.md — Промо-сайт "Городецкие призы"

## Цель
Создать промо-сайт акции, где пользователи регистрируются по номеру телефона, загружают чеки с продукцией «Городецкое молоко» для участия в розыгрыше призов.

## Цвета и дизайн бренда
Цветовая гамма — **белый, голубой, зелёный, красный**. Акцент на элементы **городецкой росписи** в дизайне. Вся вёрстка должна быть адаптивной.

## Призы
*   **Супер-приз (1-3 места)**: Экскурсия на производство + расписной сундук.
*   **Приз (4-15 места)**: Экскурсия на производство + набор мерча.
*   **Поощрительный приз (16-50 места)**: Набор мерча (кружка, термос, шоппер, стикерпак).

## Этапы разработки
1.  **Этап 1:** Разработать одностраничный сайт (лендинг) с дизайном в стиле бренда.
2.  **Этап 2:** Реализовать регистрацию/логин по номеру телефона и сканер чеков.
3.  **Этап 3:** Создать эмулятор API ФНС и подключить его к сканеру.
4.  **Этап 4:** Собрать всю систему в единый работающий прототип.

## Технологический стек
*   **Бэкенд:** Node.js + Express.js.
*   **Фронтенд:** HTML, CSS, JavaScript.
*   **База данных:** PostgreSQL.
*   **Сканер:** `jsQR` или `html5-qrcode`.
